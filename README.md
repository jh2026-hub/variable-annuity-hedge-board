# 변액 헤지 보드 — 감마가 매년 얼마를 가져가나

변액보증(GMxB) 헤지에서 **감마(볼록성) 누수**가 왜, 얼마나 손실로 이어지는지를 보여주는 인터랙티브 설명 도구입니다. 델타·금리 헤지는 프리미엄이 없어 거의 공짜지만, 그 순간 따라붙는 감마는 시장이 움직이기만 하면 방향과 무관하게 손실을 내고 그 크기는 변동성의 제곱(½·Γ·σ²)에 비례합니다. 이 도구는 보증수수료라는 재원 안에서 헤지비용과 감마 누수가 어떻게 소진되는지, 그리고 나쁜 해·최악의 해에 손실이 CSM 완충을 넘어 보험손익으로 터지는 과정을 시각화합니다.

**라이브:** https://juhyeonwork-design.github.io/variable-annuity-hedge-board/

## 조작법

- **리스크 카드 토글** — 각 카드를 눌러 헤지 on/off. 그룹1(방향)은 켤수록 흔들림이 줄고, 그룹2(감마)는 델타·금리 헤지를 켜야 활성화되며 켜면 프리미엄 확정·끄면 매년 누수. 그룹3(양방향)은 대개 감내.
- **감마 계산기** — 주가/금리 감마의 Γ·실현변동성(RV)·내재변동성(IV)을 직접 입력하면 "감내 시 새는 손실" vs "헤지 시 프리미엄"이 즉시 계산됩니다. (모바일 숫자 키패드 지원)
- **CSM 시나리오 버튼** — 완충장치 CSM 잔액(얇음/보통/두터움)을 바꿔 손실이 보험손익으로 터지는 임계를 확인.
- **세부 가정 바꾸기** — 하단 토글에서 CSM·보증수수료·적립금·잔여 변동성을 슬라이더로 조정.

## 범용 도구 · 주의

**적립금·보증수수료·CSM·감마·변동성 등 모든 값은 직접 입력할 수 있는 변수입니다** — 자기 회사·포트폴리오 숫자를 넣어 범용으로 쓰세요. 화면에 보이는 기본값은 개념 설명용 예시일 뿐 특정 회사·실제 포지션과 무관합니다. 계산식은 교육용으로 단순화된 근사이며 실제 헤지 손익 평가에 사용해서는 안 됩니다.

## 기술

빌드 도구 없음. 순수 HTML/CSS/JS 단일 파일(`index.html`)이라 그대로 서빙합니다. GitHub Pages(main 브랜치 루트)로 배포.
